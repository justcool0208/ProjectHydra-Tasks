# Task 1-A: Introductory ML Learning Path (Optional)

**Objective:** This repository contains a curated sequence of tasks designed to give you an introduction to Machine Learning.
Tasks include hands-on coding, conceptual understanding, and implementing ML models from scratch and with libraries such as NumPy, Pandas, Matpltlib, and PyTorch.
